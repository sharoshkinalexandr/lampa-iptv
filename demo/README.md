# Браузерное демо

Демо запускает production-сборку `plugin.js` поверх минимального mock API Lampa. Оно предназначено
для проверки загрузки, меню, Activity, фокуса, категорий, карточек и настроек источников.

Реальные трансляции, учётные данные и каналы 18+ в стенде не демонстрируются. Mock-плеер только
показывает, что получил запрос на воспроизведение.

После `npm run build` откройте каталог `site` любым статическим HTTP-сервером и перейдите на
`/demo/`. Например:

```powershell
npx http-server site -p 4173 -c-1
```
