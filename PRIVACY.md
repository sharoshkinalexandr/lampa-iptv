# Конфиденциальность

Плагин не содержит телеметрию, рекламу, аналитику, удалённый аккаунт или собственный backend.

Локально могут храниться:

- пользовательские URL M3U/XMLTV и прямых потоков;
- Xtream server, login и password;
- HTTP-заголовки источников;
- избранное, история, последний канал и UI-настройки;
- соль/хеш PIN и временная отметка разблокировки;
- кеш XMLTV и результаты локальной диагностики.

Данные отправляются только на адреса, которые пользователь явно добавил, и в сам Lampa Player.
Единственный штатный запрос проекта — суточная проверка `version.json` на том же origin, откуда
загружен плагин.

GitHub Actions не получает доступ к browser storage. Автоматическая проверка использует только
публичный allowlist из репозитория; приватные источники туда добавлять запрещено.

Для удаления данных используйте сброс в настройках плагина, затем удалите URL расширения. Полную
очистку можно выполнить через удаление данных сайта/WebView Lampa.
