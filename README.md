# Lampa IPTV

Production-плагин IPTV для Lampa со строгим реестром из **ровно 28 заданных каналов**. Плагин не
подмешивает общий публичный каталог: импортированные M3U и Xtream-записи только сопоставляются с
утверждённым списком, а всё остальное попадает в отчёт как проигнорированное.

> В репозитории нет встроенных телевизионных потоков, приватных логинов или токенов. Добавьте
> легальный M3U/XMLTV своего провайдера либо официальный прямой HTTP(S)-поток самостоятельно.

## Возможности

- один production-файл `plugin.js`, загружаемый в Lampa без npm на телевизоре;
- 28 фиксированных каналов, категории, поиск, избранное и история;
- удалённый M3U, вставка текста, файл M3U, прямая ссылка и Xtream Codes;
- строгий матчинг по `tvg-id`, `tvg-name`, названию, псевдонимам и ручным правилам;
- до трёх источников на канал и автоматический резерв через Lampa Player;
- XMLTV/XMLTV.GZ, текущая и следующая программа, смещение времени и локальный кеш;
- клиентская диагностика ссылок с раздельными статусами HTTP, CORS и авторизации;
- локальный PIN для раздела 18+; взрослые каналы скрыты из поиска, истории и демо по умолчанию;
- безопасный экспорт без паролей, токенов, cookies, Authorization и Xtream credentials;
- управление фокусом через Controller Lampa и клавиатурой/пультом;
- GitHub Actions для CI, Pages, безопасной проверки только разрешённых публичных источников и
  релиза с SHA-256.

## Установка

После публикации используйте точный установочный URL:

```text
https://sharoshkinalexandr.github.io/lampa-iptv/plugin.js
```

Добавьте его в Lampa:

1. Откройте **Настройки → Расширения → Добавить плагин**.
2. Вставьте полный HTTPS URL, оканчивающийся на `/plugin.js`.
3. Перезапустите Lampa.
4. В левом меню откройте новый пункт **ТВ**.

Страница проекта:
[sharoshkinalexandr.github.io/lampa-iptv](https://sharoshkinalexandr.github.io/lampa-iptv/).
Для локальной проверки используйте файл `dist/plugin.js`, созданный командой `npm run build`.
Файл `plugin.min.js` функционально идентичен и меньше по размеру.

## Первичная настройка

### M3U по URL

Откройте **ТВ → Источники → Удалённый M3U**, задайте название и HTTP(S) URL. Плейлист читается в
браузере устройства, поэтому сервер должен разрешать CORS. Не совпавшие с 28 каналами записи не
добавляются.

### Текст или файл M3U

В разделе **Источники** выберите вставку текста либо импорт файла. Поддерживаются BOM, CRLF/LF,
`#EXTINF`, `tvg-*`, `group-title`, `tvg-chno`, catch-up атрибуты, `#EXTGRP`,
`#EXTVLCOPT:http-*`, pipe-заголовки и относительные ссылки у удалённых плейлистов.

Минимальный пример:

```m3u
#EXTM3U url-tvg="https://epg.example.com/guide.xml"
#EXTINF:-1 tvg-id="perviy" tvg-name="Первый канал" group-title="Основные",Первый канал
https://provider.example.com/live/perviy/index.m3u8
#EXTINF:-1 tvg-id="ntv" tvg-name="НТВ" group-title="Основные",НТВ
https://provider.example.com/live/ntv/index.m3u8
```

Домены `example.com` в примере не являются рабочими потоками.

### Прямая ссылка и резервы

Откройте карточку канала без источника либо контекстное меню канала → **Настроить источники**.
Доступны основной URL, резерв №1 и резерв №2, User-Agent, Referer, внешний плеер, проверка,
перестановка приоритета и очистка.

### Xtream Codes

Откройте **Источники → Xtream Codes** и введите сервер, имя пользователя и пароль своего
легального провайдера. Учётные данные остаются в локальном хранилище устройства, маскируются в
диагностике и не входят в обычный экспорт.

### XMLTV

В настройке конкретного канала укажите:

- XMLTV или XMLTV.GZ URL;
- `channel id` / `tvg-id`;
- при необходимости часовой пояс, минутное смещение и имя-псевдоним.

EPG не обязательна: канал с настроенным видеоисточником работает без программы. Из-за CORS
удалённый XMLTV может быть недоступен браузерному WebView; в этом случае используйте XMLTV
провайдера с разрешённым доступом.

## Управление

- стрелки — перемещение;
- OK/Enter — открыть или запустить;
- Back/Escape — назад;
- долгое OK или контекстное меню — избранное, проверка, EPG, скрытие и источники.

Навигация использует Controller API Lampa. Быстрое переключение в плеере получает только
настроенные не-взрослые каналы.

## Раздел 18+

Раздел скрыт и заблокирован по умолчанию. При первом входе задайте локальный PIN из 4–8 цифр.
Хранится только соль и хеш; разблокировка действует 15 минут. Взрослые каналы не появляются в
общем поиске, истории, публичном демо, скриншотах или автоматическом запуске.

## Импорт, экспорт и сброс

**Источники → Экспорт** создаёт JSON без секретов. Экспорт с секретами должен выбираться отдельно
и храниться как конфиденциальный файл. Импорт валидирует формат, разрешённые идентификаторы и URL.

Для полного удаления:

1. удалите URL плагина в настройках расширений Lampa;
2. откройте **ТВ → Диагностика → Сбросить локальные данные**;
3. при необходимости очистите данные сайта/WebView для домена Lampa.

## Ограничения

- CORS может не позволить браузеру проверить или получить поток, хотя нативный плеер его
  воспроизводит;
- User-Agent, Referer и Cookie нельзя гарантированно установить из браузерного плеера;
- DRM, платная авторизация, CAPTCHA, Cloudflare и геоблокировки не обходятся;
- кодеки и внешний плеер зависят от устройства и сборки Lampa;
- официальный веб-плеер канала не означает, что его внутренний временный HLS разрешено
  перепубликовывать;
- плагин не является IPTV-провайдером.

## Разработка

Требуется Node.js 20 или новее.

```powershell
npm install
npm run validate
```

Артефакты появляются в `dist/`, а готовый Pages-сайт — в `site/`. Команда `npm run validate`
запускает ESLint, TypeScript, Vitest, production build, контроль размера и проверку на опасные
конструкции/секреты.

## FAQ

**Почему все карточки сначала показывают «Источник не подключён»?**

Это преднамеренно: репозиторий не распространяет телеканалы. Импортируйте свой легальный источник.

**Почему канал есть в M3U, но не появился?**

Плагин не расширяет каталог. Проверьте отчёт импорта и создайте ручное соответствие с одним из 28
каналов.

**Почему проверка говорит «CORS: неизвестно»?**

Сервер запретил браузерный запрос. Это не доказательство неработоспособности потока; попробуйте
официальное приложение или внешний плеер.

**Поддерживается ли DRM?**

Нет. Плагин не извлекает ключи и не обходит Widevine/PlayReady.
