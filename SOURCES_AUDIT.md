# Аудит источников для фиксированного каталога

Дата проверки: **2026-07-23**.

## Методика

Для каждого названия проверялись официальный сайт, официальная страница эфира и публично
подтверждаемая модель распространения. Встроить ссылку можно было только при одновременном
выполнении условий: прямой стабильный HTTP(S) HLS, явно публичное официальное распространение,
отсутствие персонального токена/логина и допустимость автоматической проверки.

Наличие официального веб-плеера не приравнивается к праву перепубликовать его внутренний URL.
Временные, подписанные, защищённые и найденные у неофициальных агрегаторов ссылки не
фиксировались. Результат: **в репозиторий не встроено ни одного телевизионного потока**; поле
`sources` у всех 28 записей пустое.

Обозначения:

- **страница** — подтверждена официальная веб-страница просмотра, но не переносимый HLS;
- **подписка** — канал распространяется через авторизованного провайдера;
- **не подтверждено** — не найден первичный официальный источник, достаточный для публикации;
- **не проверялось** для DRM/гео означает, что без допустимого прямого потока технический тест не
  выполнялся и вывод не выдумывается.

плагин принимает только локально добавленную пользователем ссылку с подтверждённым правом
просмотра.

Каналы без встроенного EPG: **все 28**. Пользовательский XMLTV можно назначить каждому каналу
отдельно.

Каналы без встроенного видеоисточника: **все 28**. Это осознанное условие безопасного релиза, а
не ошибка импорта.
